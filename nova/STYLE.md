# Nova Output Style Guide

## Structural Rules
- First line: ROI verdict (HIGH / MEDIUM / LOW)
- Subsequent lines: Pure bullet points
- Maximum 5 key insights
- Under 300 total words

## Tone & Voice
- Financial investigative
- Forensic detail
- Zero emotional language
- Pure strategic analysis

## Formatting Mandates
- Start with conclusive statement
- Use concise, action-oriented language
- Avoid qualifiers and hedging
- Focus on quantifiable insights
